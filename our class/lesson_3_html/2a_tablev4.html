Atrribute of the TD tag

1. colspan [number] use to merge colums, the higher the number the higher the column
2. rowspan [number] use to merge rows, the higher the number the higher the rows
3. align [left, right, center] set the alignment of your data
4. bgcolor [color] set the background-color
5. valign [top, bottom, center ] set the vertical alignment of the data

Atrribute of the TR tag

1. align [left, right, center] set the alignment of your data
2. bgcolor [color] set the background-color

Atrribute of the TABLE tag

1. align [left, right, center] set the alignment of your data
2. bgcolor [color] set the background-color
3. border [number] set the border for the table
4. width [number] set the width for the table
5. height [number] set the height for the table
4. cellpadding: [number] set the padding of the columns 5. cellspacing: [number] set the margin of the columns
